

## Keyboard controls

* Q - Quit
* Pause/unpause
* E - Edit Mode

### In Edit mode

* I,J,K,L - move cursor
* Space - toggle cell
* 1,2,3,4 - add a glider (in different orientations)
* G - add a glider gun
* c - clear all cells


This program is not robust. For example, it will crash if you resize the terminal window.
