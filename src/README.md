

## Keyboard controls

* Q - Quit
* Enter - Pause/Unpause
* E - Edit mode

### In Edit mode

* I,J,K,L - move cursor
* Space - toggle cell
* 1,2,3,4 - add a glider (in different orientations)
* 0 - add a glider gun
* c - clear all cells
* Enter - exit edit more


This program is not robust. For example, it will crash if you resize the terminal window.
